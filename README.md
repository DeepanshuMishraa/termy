# GPUI Terminal

Minimal GPUI-powered terminal built on gpui-rs, alacritty, and pty for a fast, clean UI.

## Run

```sh
cargo run --release
```

## Build

```sh
cargo build --release
```

## Config

Config file: `~/.config/gpui-terminal/config.txt`

```txt
# Will be comments using #
theme = tokyonight
```

Themes: `tokyonight`, `catppuccin`, `dracula`, `gruvbox`, `nord`, `solarized`.
